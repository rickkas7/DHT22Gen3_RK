Member Enumeration Documentation

◆ SampleResult

enum DHTSample::SampleResult
strong

Enumeration for result codes from the getSample() request.

Success is 0. All other codes are errors

Enumerator
SUCCESS 

Success (including valid checksum)

ERROR 

An internal error (problem with the I2S peripheral, etc.)

TOO_MANY_RETRIES 

After the specified number of retries, could not get a valid result.

BUSY 

Called getSample() before the previous call completed.
